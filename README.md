## Laravel 8 Skeleton + Nova

This is a skeleton project to kick-start your projects quicker with ready installed packages, configurations and templates!

## Requirements

**OS:** macOS or Ubuntu  
**PHP:** ^7.3 (7.4 preferred)  
**MySQL:** ^5.7.25 (not v8)  
**Redis:** ^5.0.0

#### PHP Extensions

- curl: *
- json: *
- pcntl: ^7.2
- posix: ^7.2

## Installation

1. `$ git clone git@github.com:Ince/laravel-skeleton-nova.git {project_name}`
2. `$ cd {project_name}`
3. `$ composer install -o`
4. `$ php artisan project:install`
